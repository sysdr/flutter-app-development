# nomadair_lesson_18

A self-contained Flutter project that can be pushed and run independently.

## Run This Project

From this directory (`nomadair_lesson_18`):

1. `flutter pub get`
2. `flutter analyze`
3. `flutter test`
4. `flutter run`

## Notes

- This project does not require `setup.py` to run.
- Generated/local-only files are excluded via `.gitignore` so fresh clones run cleanly.
