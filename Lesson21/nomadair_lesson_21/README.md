# nomadair_lesson_21

Standalone Lesson 21 NomadAir Flutter app.

## Run Locally

From inside `nomadair_lesson_21`:

- Install dependencies:
  - `flutter pub get`
- Run unit tests:
  - `flutter test`
- Run app on connected device/emulator:
  - `flutter run -d emulator-5554`

## Notes

- This child directory is self-contained and does not require `setup.py`.
- Do not commit generated local folders such as `.dart_tool/`, `build/`, `.idea/`.
