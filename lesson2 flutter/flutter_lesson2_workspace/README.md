# Flutter Lesson 2 Workspace

This directory contains the Lesson 2 Flutter setup and verification workspace.

## Key Files

- `setup.sh`: environment verification and workspace generator
- `cleanup.sh`: Docker/container cleanup script
- `.gitignore`: ignore rules for Flutter, Python, Node, and local artifacts
- `requirements.txt`: placeholder (no Python dependencies required)

## Cleanup

Run from this directory:

```bash
bash ./cleanup.sh
```
